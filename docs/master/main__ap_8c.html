Detailed Description

AP ( AutoPilot ) process.

This process is reponsible for the collecting the different sensors data, fusing them to obtain aircraft attitude and running the different control loops

static void attitude_loop ( void  ) [inline, static]

There are four @@@ boucles @@@:

  • 20 Hz:
    • lets use reporting_task at 60 Hz
    • updates estimator of ir with ahrs_update_infrared
    • set desired_aileron and desired_elevator with pid_attitude_loop
    • sends to fbw desired_throttle, desired_aileron and desired_elevator
      Note:
      desired_throttle is set upon GPS message reception
  • 4 Hz:
    • calls estimator_propagate_state
    • do navigation with navigation_task
